## Hangman Game

This is an Express-based Hangman game. API documentation to come!

Commands:
```bash
# Install and run
$ yarn
$ yarn start

# Run tests
# Tests should be run with `NODE_ENV=test` for test database configuration
$ yarn test
$ yarn test:unit
$ yarn test:http

# View code coverage
$ yarn test:coverage

# Lint code
$ yarn lint
```
